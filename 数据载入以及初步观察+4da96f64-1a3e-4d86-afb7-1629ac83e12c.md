1.2 pd.read_csv和pd.read_table读取：

```Python
# pd.read_csv()
```

1.3 chunk分块读取

```Python
chunker = pd.read_csv('train.csv', chunksize=1000)
#读取方式一
chunker.get_chunk()
#读取方式二
for chunk in chunker:
    print(chunk)
```
